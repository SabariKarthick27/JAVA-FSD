Task 11: Style Inspector Laboratory

Review the testing parameters to verify your CSS3 integration values using standard browser developer tools:

# dev-tools-checklist --inspect-stylesheet

1. Open Chrome's device toolbar by pressing Ctrl+Shift+M to inspect mobile viewport stacking under 768px.

2. Go to the Network panel and reload the page to confirm that the Google Fonts stylesheet and styles.css are loaded securely.

3. Right-click any .eventCard in the grid, select 'Inspect', and tweak the padding/margins in the Styles sidebar to see box model modifications instantly!
