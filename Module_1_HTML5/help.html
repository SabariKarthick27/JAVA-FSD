CityLink Portal Help Center

Welcome to the User Support and Frequently Asked Questions page. This external guide is designed to help neighborhood residents troubleshoot interactive widgets and use portal services efficiently.

Frequently Asked Questions

1. Why is the Geolocation utility failing to load coordinates?

The Geolocation API is highly secure and requires explicit user consent. When you click "Find Nearby Events", your browser will prompt you with a permission request. If you decline, or if your connection times out, a validation error is displayed. Make sure you enable location services in your browser settings.

2. How does the event pre-selection and preference saving work?

Our portal tracks your preferred event selection using the HTML5 Web Storage API. When you select an event category in the registration card, your preference is immediately logged into your local browser's storage cache. If you accidentally refresh the page, the form automatically pre-selects that event. Clicking the "Clear Saved Preferences" button completely flushes these values from both localStorage and sessionStorage.

3. How do I inspect form fields and validation errors?

You can test form validation parameters using standard browser developer tools. Simply double-click any field or right-click and choose "Inspect". The Name field is set with autofocus, and all fields are configured as required, ensuring native pop-ups guide you if critical information is missing.

4. Can I see high-resolution copies of event graphics?

Yes! In the event gallery table, you can double-click on any image thumbnail. This triggers a modal overlay enlarging the illustration so you can inspect event posters closely.

Need Extra Help?

If you encounter rendering issues or technical glitches, please contact the City Council Support Office or check out our active logs via the browser console.
